Bug Tracker / Project Management App

Bug tracker is a web app I created as a way of learning the MERN (MongoDB, Express.js, React.js, Node.js) stack. It is a full-stack web app with a frontend SPA, and a backend web service. I created this from January to February 2023 as I learned the previously mentioned stack. The app doubles as a project management app, and features users, projects, tickets, notes, roles, role-based permissions, authentication, a user dashboard which displays a user's projects and tickets, light/dark mode and more. The project is hosted on render.com and can be demoed here. (NOTE: Render.com takes a minute to load when first accessing the site)

St. Paul Crime Incidents Tracker

The St. Paul Crime Incidents web app is a full-stack project I worked on in a team of 3 as part of my web development course. The app dynamically displays crime incidents throughout St. Paul on an interactive map. Incidents can be filtered by incident/crime type, neighborhood, and date & time. Users can also add and delete new crimes to and from the database. The app is written in Vue.js and uses Leaflet and the Nominatim API for the interactive map and street addresses. It retrieves data from a RESTful API created with Node.js Express which pulls data from the St. Paul Crime Incident Report database. Use the tracker here.

Zoom Meeting SDK Integration

As part of my Senior Capstone course at St. Thomas, I worked in a team of 6 to integrate the Zoom Meeting SDK into a SPOCK (small private online course keeper) web app. This project was completed using Java, JSP, Spring, and Hibernate. I was involved in researching and studying the Zoom Meeting SDK and API as well as OAuth 2.0 authorization process. This project was completed using agile development practices, where we worked in sprints to implement and test features. Check out the demo here.
